# 40-Mile Ultra Loop 🚴🏃

**Advanced only.** Full benchmark loop combining the Forest Park perimeter, Skyline ridgeline, St. Johns Bridge descent, downtown connector, and return climb. The definitive big-loop training route for serious athletes.

## Route Details

| Detail | Value |
|--------|-------|
| **Distance** | ~40,000 m (~25 mi) |
| **Elevation Gain/Loss** | ~2,500 ft total |
| **Est. Cycling Time** | 3–4 hrs |
| **Est. Running Time** | 10–14 hrs |
| **Surface** | Road + trail mix |
| **Difficulty** | Advanced |

### Waypoints (Clockwise)
1. **Start:** Ridge Trail Parking Lot — 45.588, -122.794
2. **Ridge:** NW Skyline Blvd → panoramic ridgeline
3. **North:** NW St. Helens → NW Germantown
4. **Cross:** NW Yeon → NW Going → NW Fairview
5. **South:** NW Warrenton → NW Cornell → NW Lovejoy
6. **Fleet:** NW Folsom → NW Thurman
7. **Descent:** St. Johns Bridge (45.593, -122.747)
8. **Waterfront:** Willamette River — 45.522, -122.672
9. **Return:** NW Marshall → NW Lovejoy → NW 23rd
10. **Climb:** Back up to Ridge via NW Fairview / NW Cornell

### Description
This is the big loop. Starting at Ridge Trail, ride north on Skyline Blvd along the ridgeline, then drop down to the park's northern road network (St. Helens, Germantown). Cross the western section via Yeon and Going, then descend through the residential streets. Cross the St. Johns Bridge into downtown, finish at the waterfront, and climb back.

### Highlights
- **Complete Circuit:** Does what no other Forest Park route does — full loop
- **Varied Terrain:** Ridgeline climbing, forest canopies, river waterfronts
- **Training:** Unmatched for marathon/Triathlon prep
- **Landmark:** St. Johns Bridge, Pittock Mansion, Mt. Hood views

### Notes
- **3–4 hours for cycling, 10–14 for running** — plan accordingly
- Carry water (no reliable fountains on the route)
- St. Johns Bridge cycle path is narrow — single file
- Mountain bike recommended for jungle sections of the loop
- Runtime for running: plan 10–14 hrs with walk breaks
