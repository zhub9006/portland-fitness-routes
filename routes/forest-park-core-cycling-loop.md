# Forest Park Core Cycling Loop

**Type:** Cycling
**Difficulty:** Easy to Moderate
**Best Season:** Spring–Fall

## Route: Forest Park Core Cycling Loop

| Metric | Value |
|--------|-------|
| **Full Loop Distance** | 11,307 m (~7.0 mi) |
| **Duration** | ~25 min biking |
| **Elevation** | ~100 ft (mostly flat, gentle rolling) |
| **OSM Verified** | ✅ Bike routing via NW Saint Helens Rd, NW 25th Ave, NW Cornell Rd |

**Coordinates:**
- Forest Park Center: 45.536, -122.739
- Bridge Ave & St. Helens: 45.557, -122.770

**Route Segments:**
| Segment | Distance | Road |
|---------|----------|------|
| Forest Park Center → NW St. Helens Rd | ~2,500 m | NW Cornell Rd → NW St. Helens Rd |
| NW St. Helens Rd → Bridge Ave | ~3,300 m | NW St. Helens Rd → NW Bridge Ave |

**Highlights:**
- 🌲 Complete cycling loop through Forest Park
- 🚴‍♂️ Well-maintained roads with bike-friendly surfaces
- 🔄 Easy to ride in either direction
- 📍 Multiple bailout points if you want a shorter ride

**Running Club Tips:**
- Best done on Sundays when St. Helens section is car-free
- Pair with the Ridge Loop for a ~15 km combined ride
- Ideal warm-up before a longer ride or trail run
- Weekday mornings: minimal traffic on all segments
