# Saltzman Warmup Loop

## Overview

- **Type:** Running / Walking / Cycling
- **OSM Verified Distance:** 4,586 m (full loop); 3,764 m (St. Helens segment)
- **Duration:** ~7 min biking (full loop); ~4 min biking (St. Helens segment)
- **Difficulty:** Easy
- **Elevation Gain:** ~150 ft
- **Surface:** Paved road

## Start Point

- **Coordinates:** 45.564, -122.757
- **Location:** NW Kittridge Ave & NW Saint Helens Rd (north entrance area)

## Route Description

A tight, fast warmup loop near Forest Park's north entrance. Perfect for speed work, track-style intervals, and rolling-effort tempo rides. The route uses small residential connector streets to form a compact loop that keeps you in the park vicinity without long tours.

### Full Loop

```
Start: NW Kittridge Ave & NW Saint Helens Rd (45.564, -122.757)
→ NW Saint Helens Rd (2,837 m)
→ NW Bridge Avenue (1,923 m)
→ NW Saltzman Road (1,192 m)
→ NW Nicolai Street (107 m)
→ NW 30th Avenue (109 m)
→ NW Wilson Street (273 m)
→ NW 33rd Avenue (80 m)
→ NW Vaughn Street (99 m)
→ NW 32nd Avenue (160 m)
→ NW Franklin Court (280 m)
→ NW Aspen Avenue (429 m)
→ NW Belgrave Avenue (370 m)
→ Return to start
Distance: 4,586 m (~1.5 mi)
```

### St. Helens Road Segment (Out-and-Back)

```
Start: NW Saint Helens Rd & NW Bridge Ave (45.557, -122.770)

→ Turnaround at NW Front Avenue
→ Return south
Distance: 3,764 m (one-way)
```

## Highlights

- 🔒 **Tight Loop:** Compact route perfect for intervals and speed work
- 🌲 **Park Adjacent:** Keeps you in the Forest Park vicinity
- 🚗 **Minimal Traffic:** Residential streets, low vehicle volume
- ⚡ **Fast Surface:** Smooth paved roads with good bike lines
- 📊 **Interval Training:** Perfect for track-style workouts outdoors

## Running Club Notes

- **Best for:** Warmup/cooldown, intervals, tempo runs
- **Surface:** Paved — use road shoes or hybrid tires
- **Caution:** Some intersections require careful crossing (Trail etiquette)
- **Time:** 10-15 minutes for a full loop cycle, 7-8 min walk
- **Group Rides:** Excellent for group ride setups — loop and restart
