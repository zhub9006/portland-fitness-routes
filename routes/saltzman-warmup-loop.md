# Saltzman Warmup Loop

**Type:** Cycling  
**Difficulty:** Easy  
**Best Season:** Year-round

---

## Route: Saltzman Road + St. Helens Road + Bridge Avenue Circuit

| Metric | Value |
|--------|-------|
| **Full Loop Distance** | 4,586 m (~2.9 mi) |
| **St. Helens Segment** | 3,764 m (~2.3 mi) one-way |
| **Duration** | ~4.7 min biking (full loop) |
| **Elevation** | ~300 ft (gentle) |
| **OSM Verified** | ✅ Bike routing via Saltzman Rd → NW St. Helens Rd → NW Bridge Ave |

**Coordinates:**
- St. Helens Rd Start: 45.535, -122.739
- Bridge Ave connector: 45.5769, -122.7614

**Route Segments:**
| Segment | Distance | Road |
|---------|----------|------|
| Saltzman Rd → NW St. Helens Rd | ~1,200 m | NW St. Helens Rd (westbound) |
| NW St. Helens Rd → Bridge Ave | ~1,777 m | NW St. Helens Rd → NW Bridge Ave |
| Bridge Ave → Return via NW St. Helens Rd | ~1,609 m | NW Bridge Ave → NW St. Helens Rd (eastbound) |

**Highlights:**
- ⚡ Tight, fast loop — perfect for speed work and track-style intervals
- 🏃‍♂️ Great warm-up before hitting trails or longer rides
- 🌲 Quiet residential streets with park access
- 🔄 Simple counterclockwise loop

**Running Club Tips:**
- 3.5-mile time-trial: hit it in under 10 min
- Pair with Leif Erikson for a 6+ mile warm-up
- Ideal for Tuesday/Thursday run club group rides
- Alternate direction for variety
