# Saltzman Warmup Loop

**Type:** Cycling | **Category:** Warm-Up / Speed Work

## Route: Saltzman Warmup Loop
- **Verified Distance:** 4,586 m full loop; 3,764 m (St. Helens segment)
- **Duration:** ~4 min bike on St. Helens segment
- **Elevation Gain:** ~300 ft
- **Surface:** Residential streets + park connector
- **Surface Notes:** Paved, tight turns

### Turn-by-Turn
1. Start at NW Saint Helens Road & NW Saltzman area
2. Loop via NW Saltzman, NW Miller, NW Germantown Rd
3. Connect to NW Saint Helens Road for the main segment
4. Complete the full loop back to start

### Highlights
- Tight, fast loop near the north entrance to Forest Park
- Great for speed work and track-style intervals
- Short enough to do multiple laps
- Good warm-up before longer rides
- Ideal for cadence work and sprint intervals

### Training Use
- **Speed work:** Multiple laps at target effort
- **Warm-up:** 3-4 laps before a ride
- **Endurance:** Extended loops combining with Ridge Loop
