# Germantown Ridge Loop (Cycling)

## Overview

- **Type:** Cycling
- **OSM Verified Distance:** 4,586 m (full loop); ~7,240 m (extended circuit)
- **Duration:** ~7 min biking (full loop); ~12 min biking (extended)
- **Difficulty:** Easy-Moderate
- **Elevation Gain:** ~200 ft
- **Surface:** Paved road

## Start Point

- **Coordinates:** 45.557, -122.770
- **Location:** NW Bridge Avenue (Ridge Trail Tom St)

## Route Description

This warmup loop circumnavigates the Germantown Ridge, one of Forest Park's many forested ridges. The route follows quiet residential streets and trail shoulders for a scenic, low-stress ride. It's an excellent pre-strip or finish-line loop.

### Full Ridge Circuit

```
Start: NW Bridge Ave (45.557, -122.770) / NW Gravity Loop
→ NW Wildwood Trail (on-road segment) (809 m)
→ NW Saint Helens Rd (2,837 m)
→ NW Germantown Rd (3,356 m)
→ NW Kaiser Road / NW Laidlaw Rd (connection)
→ Return to start
Distance: 4,586 m (~1.5 mi)
```

### Extended Circuit (Panoramic)

```
Start: Forest Park center (45.536, -122.739)
→ NW Cornell Rd (climb) (2,798 m)
→ NW Lovejoy St ( connector) (1,389 m)
→ NW 23rd Ave ( connector) (741 m)
→ NW Thornton St (1,730 m)
→ NW Johnson St (793 m)
→ NW Germantown Rd (3,356 m)
→ NW Skyline Blvd (4,529 m)
→ Return to start
Total: 14,636 m (~9 mi)
```

## Highlights

- 🌄 **Germantown Ridge:** Sweeping views of the city skyline
- 🌲 **Residential Cool:** Quiet streets with tree-lined canopy
- 🔁 **Repeat Options:** Easy to loop multiple times for interval sets
- 🚸 **Road Safety:** Minimal traffic; visible road markings
- 🏁 **Finish Line:** Great for group rides to end with a sprint

## Running Club Notes

- **Best for:** Speed work, warmup, cooldown
- **Loop Count:** 3-5 loops for a 10-15 min interval workout
- **Connects:** Links to Wildwood Trail for combined run+ride
- **Wind:** Ridge can be breezy — handlebar-mounted wind meter recommended
