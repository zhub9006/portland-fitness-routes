# OSM-Verified Routes Overview

This page provides a comprehensive overview of all Forest Park routes that have been verified using OpenStreetMap (OSM) path and road data. Distances and durations were computed using the OSM routing engine.

## How Routes Are Verified

Each route follows actual paths and roads recorded in OpenStreetMap (OSM). The routing engine calculates the precise distance and estimated walking/cycling time based on:
- Path type (paved, gravel, dirt, trail)
- Elevation changes
- Surface conditions
- Turn restrictions

## Key OSM Data Sources
- **Wildwood Trail** — OSM leisure=path, highway=path
- **Leif Erikson Drive** — OSM highway=secondary (car-free on Sundays)
- **NW Skyline Blvd** — OSM highway=secondary (rural road)
- **NW Saint Helens Rd** — OSM highway=tertiary
- **NW Cornell Rd** — OSM highway=tertiary
- **Residential connecting streets** — OSM highway=residential
