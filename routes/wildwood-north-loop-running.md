# Wildwood Trail North Loop Running Route 🏃

**Route Type:** Running | **Difficulty:** Moderate | **Elevation:** ~800 ft total

## Route Overview

The Wildwood Trail North Loop connects the Wildwood Trail, Leif Erikson Drive, and the Germantown Ridge loop into one continuous circuit. This is the ultimate Forest Park run — it combines the immersive forest experience of the Wildwood Trail with panoramic ridge views and a scenic Descent back to the park base.

## Highlights

- 🌲 **Wildwood Trail** — 5+ km through old-growth Douglas fir and western red cedar, one of the best urban trail runs in America
- 🌿 **Moss & ferns** — The moss-draped canopy section is quintessentially Pacific Northwest
- 🌉 **Leif Erikson Drive** — Car-free on Sundays; paved and smooth for efficient running
- 🏔️ **Ridge loop** — Kaiser to Laidlaw offers the best in the park views
- 🌊 **Yeon descent** — Fast descent through the forest to the river
- 🦫 **Creek crossings** — Multiple seasonal creeks to splash through (spring/early summer)
- 🍂 **Fall foliage** — The Wildwood Trail in autumn is absolutely spectacular
- 🦅 **Wildlife** — Owls, pileated woodpeckers, and deer are common on the loop

## How to Use This Route

### Run 1: Wildwood Core Loop (1.5 hrs)
1. Start at **Lower Macleay Park** (45.533, -122.713)
2. Run the Wildwood Trail northbound → 3,000 m
3. Return via the Looper exit to **Leif Erikson Drive** → 500 m
4. Take **Kaiser → Laidlaw → Saint Helens** descent
5. Return to Macleay via **Wildwood Trail southbound**
6. **Total:** ~6.8 mi with ~800 ft of climbing

### Run 2: Extended Forest Park Loop (2.5 hrs)
1. Complete the Wildwood North Loop as above
2. After returning to Macleay, take **Cornell Road → Westover → Cumberland → Albemarle → Beuhla Vista → Hermosa**
3. Continue to **Skyline Blvd → Ridge Loop → Saint Helens → Yeon → Downtown**
4. **Total:** ~9.2 mi / full perimeter with Wildwood Trail extension

### Run 3: Sprint Intervals (45 min)
1. Start at **Ridge Trail Parking Lot** (45.588, -122.794)
2. Sprint up **Kaiser Road** to the ridge → 700 m
3. Sprint down **Laidlaw Road** → 800 m
4. Walk recovery on **Saint Helens Road** → 1,777 m
5. Repeat the Kaiser-Laidlaw circuit 3-4 times
6. Finish with a **Wildwood Trail cool-down** → 3,000 m

## Route Notes

- ⚠️ **Trail footing** — Wildwood Trail is rooty, rocky, and can be slippery when wet
- ⚠️ **Trail etiquette** — Hikers have right-of-way on narrow trail; ring a bell or call out when passing
- 🌧️ **Mud** — Wildwood Trail gets muddy in winter/spring; consider traction devices
- 🔦 **Early mornings** — Tree canopy is thick; it's darker than it looks in the forest
- 🧗 **Technical sections** — Wildwood has some steep section with exposed tree roots
- 🐕 **Dogs** — Allowed on leash on trails; many runners run with dogs
