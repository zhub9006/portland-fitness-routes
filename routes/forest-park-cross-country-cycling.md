# Forest Park Cross-Country Cycling Route

**Type:** Cycling
**Difficulty:** Moderate
**Best Season:** Spring–Fall

---

## Route: Full Cross-Country Forest Park Cycling

| Metric | Value |
|--------|-------|
| **Distance (Full Loop)** | 15,028 m (~9.3 mi) |
| **Distance (Center → Downtown)** | 9,440 m (~5.9 mi) |
| **Duration (Full)** | ~30 min biking |
| **Elevation** | ~500 ft total (descent to downtown) |
| **OSM Verified** | ✅ Bike routing via St. Johns Bridge, Marine Dr, St. Helens Rd, Burnside St, Naito Pkwy |

**Coordinates:**
- Forest Park Center: 45.536, -122.739
- St. Johns Bridge: 45.593, -122.747
- Willamette River waterfront: 45.522, -122.672

**Route Segments:**
| Segment | Distance | Road |
|---------|----------|------|
| Forest Park Center → St. Helens Rd | ~2,500 m | NW Cornell Rd → NW St. Helens Rd |
| NW St. Helens Rd → St. Johns Bridge (north) | ~6,709 m | NW St. Helens Rd → Bridge Ave → Lombard St |
| St. Johns Bridge → Downtown (south via Marine Dr) | ~5,810 m | Lombard St → Marine Dr → Burnside St → Naito Pkwy |

**Highlights:**
- 🌉 Start at St. Johns Bridge — iconic 1931 suspension bridge
- 🌊 End at the Willamette River waterfront in downtown Portland
- 📉 Long downhill run from the bridge into downtown
- 🔄 Combine with the Core Cycling Loop for a ~26 km adventure ride

**Running Club Tips:**
- This is a big ride — start with a proper warm-up at the Field House
- The St. Johns Bridge to downtown section is a thrilling descent — check brakes!
- Reverse direction: climb from downtown through Forest Park for a serious workout
- Pair with the core loop for a 15+ km group ride
- Best done on weekends when downtown is less congested
